# zijinscanner
### 紫金扫码 App. Based on Flutter.

### 创建命令

zijinscanner

``` flutter
flutter create --org com.rtzl zijinscanner -a java
```



### 签名密钥

```
keytool -genkey -v -keystore ~/zijinscanner-keystore.jks -keyalg RSA -keysize 2048 -validity 30000 -alias zijinscanner
```
